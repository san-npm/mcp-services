Connect via MCP

Add this to your agent's MCP config.

mcp config:
{
  "mcpServers": {
    "mcp-services": {
      "url": "https://mcp.skills.ws/mcp/sse"
    }
  }
}

REST API

Or call tools directly over HTTP.

  • /health
  • Web Scraping
  • /api/scrape?url=<url>
  • /api/crawl?url=<url>&depth=2&maxPages=10
  • /api/extract?url=<url>
  • SEO Toolkit
  • /api/serp?keyword=<keyword>
  • /api/onpage-seo?url=<url>
  • /api/keywords?keyword=<keyword>
  • Agent Memory
  • POST /api/memory {namespace, key, value, tags}
  • GET /api/memory?namespace=&key=
  • GET /api/memory/search?namespace=&query=
  • GET /api/memory/list?namespace=
  • DELETE /api/memory?namespace=&key=
  • Content & Media
  • /api/screenshot?url=<url>&format=png|jpeg
  • /api/pdf?url=<url>
  • /api/html2md?url=<url>
  • /api/ocr?url=<image_url>
  • Domain Intelligence
  • /api/whois?domain=
  • /api/dns?domain=&type=ALL
  • /api/ssl?domain=
  • Blockchain
  • /api/chain/balance?address=&chain=ethereum
  • /api/chain/erc20?address=&token=&chain=
  • /api/chain/tx?hash=&chain=ethereum

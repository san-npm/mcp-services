Connect via MCP

Add this to your agent's MCP config.

mcp config:
{
  "mcpServers": {
    "mcp-services": {
      "url": "https://mcp.skills.ws/mcp/sse"
    }
  }
}

REST API

Or call tools directly over HTTP.

  • /health
  • /api/screenshot?url=<url>&format=png|jpeg
  • /api/pdf?url=<url>
  • /api/html2md?url=<url>
  • /api/ocr?url=<image_url>
  • /api/whois?domain=
  • /api/dns?domain=&type=ALL
  • /api/ssl?domain=
  • /api/chain/balance?address=&chain=ethereum
  • /api/chain/erc20?address=&token=&chain=
  • /api/chain/tx?hash=&chain=ethereum

FAQ

Free is great for trying tools or light use (up to 10 calls/day). Pro is best for teams or workflows that need unlimited access with a predictable monthly cost. Pay-Per-Call is ideal for unpredictable usage — you only pay for what you use via crypto micropayments.

Pay-Per-Call uses the x402 protocol and supports USDC and USDT stablecoins on Base and Celo. Both are low-cost chains with fast finality, so transaction fees are minimal.

x402 is an open protocol for HTTP-native micropayments. When you make an API call, the server responds with a 402 Payment Required status. Your client automatically sends a small crypto payment and retries. No accounts, no API keys — just pay and use.

Yes. Upgrade from Free to Pro anytime. Switch from Pro to Pay-Per-Call by cancelling your subscription and using x402 payments. No lock-in.

Pro subscriptions can be cancelled anytime — you keep access until the billing period ends. Pay-Per-Call transactions are final (on-chain micropayments), but at $0.005/call the risk is minimal.
